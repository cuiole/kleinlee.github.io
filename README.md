# 最简数字人
# 移动端实时对话 无需服务器  
Notes：目前仅支持对接阿里百炼平台的ASR、LLM和TTS，数字人和vad皆为网页计算。让你可以在一分钟内零成本部署一个对话数字人。
IOS仅支持IOS17以上系统。

开源项目前身为DH_live [here](https://github.com/kleinlee/DH_live)。

商业化网页应用：[matesx.com](matesx.com), 你可以直接打开网页查看完整体应用。
